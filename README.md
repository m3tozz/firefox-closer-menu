# ☮️ firefox-closer-menu

Tab Closer is a simple Firefox extension that helps you manage your browser tabs and windows efficiently. With this tool, you can quickly close:

- Other tabs  
- Tabs to the left  
- Tabs to the right  
- Other Firefox windows
- All tabs
